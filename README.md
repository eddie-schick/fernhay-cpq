
## SHAED CPQ v1.5 - Frontend

### Setup

- open terminal on root project directory
- `cd frontend`
- `pnpm install`
- create .env file in frontend directory
- add all the variable to the .env file, example values are in .env.example
- `pnpm run dev`
- application will be running on `http://localhost:5173`
- `pnpm run build` to build the project and `pnpm run preview` to run the project in production mode
